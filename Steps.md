# Step-1
## 🏗️ Game Structure

We will build it in layers:

### 🟢 Phase 1: World (NOW)
- College map  
- Buildings  
- Roads  
- Ground  
- Basic lighting  

👉 **Goal:** “Walkable real college”

---

### 🟡 Phase 2: Simulation Mode
- Free roam mode  
- Interact with objects (doors, chairs, etc.)  
- NPC students (optional later)  

👉 **Goal:** “Real campus experience”

---

### 🔴 Phase 3: Game Mode (Escape / Missions)
- Locked doors  
- Objectives (find key, go to class, escape campus)  
- Timer or stealth mechanics  

👉 **Goal:** “College Escape style game”
